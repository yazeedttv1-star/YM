01 مبتدئ

ما هو HTML؟ وكيف يعمل؟

HTML هو اختصار لـ HyperText Markup Language، وهو الهيكل الأساسي لأي صفحة ويب. فكّر فيه كـ"عظام الجسم" – يحدد ما يوجد في الصفحة وترتيبه، لكن بدون تصميم بعد. المتصفح يقرأ ملف HTML ويحوله إلى محتوى مرئي أمامك.

HTML 
<!DOCTYPE html>
<html lang="ar">
  <head>
    <meta charset="UTF-8">
    <title>صفحتي الأولى</title>
  </head>
  <body>
    <!-- هنا يظهر المحتوى للمستخدم -->
    <h1>مرحبا بالعالم!</h1>
  </body>
</html>
💡 كل ملف HTML يبدأ بـ <!DOCTYPE html> ليُخبر المتصفح بإصدار HTML المستخدم.
02 أساسي

العناصر والوسوم الأساسية

HTML يتكون من وسوم (Tags) تُغلف المحتوى. معظم الوسوم لها وسم فتح وإغلاق. العناوين من <h1> إلى <h6>، والفقرات <p>، والروابط <a>، والصور <img> من أهم عناصر HTML التي ستستخدمها يومياً.

HTML 
<!-- العناوين -->
<h1>عنوان رئيسي (الأكبر)</h1>
<h2>عنوان ثانوي</h2>
<h3>عنوان ثالثي</h3>

<!-- الفقرة والرابط والصورة -->
<p>هذه فقرة نصية عادية.</p>

<a href="https://example.com">
  اضغط هنا
</a>

<img
  src="photo.jpg"
  alt="وصف الصورة"
>
💡 وسم <img> لا يحتاج وسم إغلاق – يُسمى وسماً مفرداً (Self-closing tag).
03 تطبيقي

بناء نموذج HTML كامل

الآن نطبّق: سنبني هيكل صفحة ويب حقيقية تحتوي على قسم ترحيبي، قائمة تنقل، وقسم محتوى. سنستخدم وسوم التخطيط الدلالية مثل <header>، <nav>، <main>، و<footer> لهيكل احترافي وصديق لـ SEO.

HTML – صفحة كاملة 
<body>

  <header>
    <nav>
      <a href="/">الشعار</a>
      <ul>
        <li><a href="#about">عنّا</a></li>
        <li><a href="#contact">تواصل</a></li>
      </ul>
    </nav>
  </header>

  <main>
    <section id="about">
      <h1>مرحباً بك!</h1>
      <p>هذه صفحتي الأولى.</p>
    </section>
  </main>

  <footer>
    <p>© 2025 اسمي</p>
  </footer>

</body>
💡 استخدام الوسوم الدلالية مثل <main> و<footer> يُحسّن ترتيب موقعك في محركات البحث.
01 مبتدئ

ما هو CSS وكيف يعمل؟

CSS هو اختصار لـ Cascading Style Sheets، وهو اللغة المسؤولة عن مظهر صفحات الويب وتصميمها. إذا كان HTML هو العظام، فـ CSS هو الجلد والملابس. يمكنك تغيير الألوان، الخطوط، المسافات، وحتى إضافة حركات رائعة باستخدام CSS فقط.

CSS 
/* تنسيق العنوان الرئيسي */
h1 {
  color: #FF6B35;
  font-size: 2.5rem;
  font-weight: 700;
  text-align: center;
}

/* تنسيق الفقرات */
p {
  color: #cccccc;
  font-size: 1rem;
  line-height: 1.8;
}
💡 "Cascading" تعني أن القواعد تتدفق من الأعلى إلى الأسفل – القاعدة الأخيرة تتغلب على السابقة إذا تعارضتا.
02 أساسي

Flexbox – التخطيط المرن

Flexbox هو نظام CSS لترتيب العناصر في صف أو عمود بكل مرونة. أصبح المعيار الأساسي لبناء القوائم والبطاقات والهيدر. بخاصية واحدة display: flex تتحكم في توزيع العناصر بشكل احترافي.

CSS – Flexbox 
/* حاوية Flex */
.nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 1rem;
}

/* بطاقات بجانب بعضها */
.cards {
  display: flex;
  flex-wrap: wrap;
  gap: 1.5rem;
  justify-content: center;
}

/* كل بطاقة */
.card {
  flex: 1 1 280px;
  max-width: 360px;
}
💡 flex-wrap: wrap يجعل العناصر تنزل إلى سطر جديد تلقائياً في الشاشات الصغيرة.
03 تطبيقي

تصميم بطاقة احترافية من الصفر

سنبني بطاقة (Card) كاملة مع صورة، عنوان، نص، وزر – بكل تأثيراتها. سنستخدم border-radius للحواف المدوّرة، وbox-shadow للظل، وtransition لتأثير Hover ناعم يرفع البطاقة عند التمرير عليها.

CSS – Card Component 
.card {
  background: #1a1a2e;
  border-radius: 16px;
  overflow: hidden;
  box-shadow: 0 4px 20px rgba(0,0,0,0.3);
  transition: transform 0.3s ease,
              box-shadow 0.3s ease;
}

.card:hover {
  transform: translateY(-8px);
  box-shadow: 0 16px 40px rgba(0,0,0,0.5);
}

.card img {
  width: 100%;
  height: 200px;
  object-fit: cover;
}

.card-body {
  padding: 1.5rem;
}

.card-btn {
  display: inline-block;
  padding: 0.6rem 1.4rem;
  background: #4ECDC4;
  color: #000;
  border-radius: 8px;
  font-weight: 600;
  text-decoration: none;
}
💡 object-fit: cover يجعل الصور تملأ حاويتها بدون تشويه حتى لو أبعادها مختلفة.
01 مبتدئ

ما هو JavaScript ولماذا نحتاجه؟

JavaScript هي لغة البرمجة الوحيدة التي يفهمها المتصفح بشكل مباشر. تُضيف التفاعل والحركة والمنطق للصفحة. بها تصنع القوائم المنسدلة، تحقق من النماذج، تحمّل البيانات بدون إعادة تحميل الصفحة، وتبني تطبيقات كاملة.

JavaScript 
// المتغيرات – طريقة تخزين القيم
const name = "أحمد";
let   age  = 25;
var   city = "الرياض"; // قديم، تجنّبه

// إظهار قيمة في الكونسول
console.log(name);      // "أحمد"
console.log(age + 5);   // 30

// رسالة تنبيه للمستخدم
alert("مرحباً بك يا " + name);
💡 استخدم const دائماً إذا القيمة لن تتغير، وlet إذا ستتغير. تجنّب var في الكود الحديث.
02 أساسي

التحكم في DOM – تغيير الصفحة حياً

DOM اختصار لـ Document Object Model، وهو تمثيل الصفحة كشجرة من العناصر. JavaScript تتيح لك الوصول إلى أي عنصر وتغييره – نصه، لونه، حجمه، إخفاءه، أو إضافة عناصر جديدة لحظياً بدون إعادة تحميل الصفحة.

JavaScript – DOM 
// اختيار عنصر من الصفحة
const title = document.querySelector("h1");
const btn   = document.getElementById("myBtn");

// تغيير النص واللون
title.textContent = "عنوان جديد!";
title.style.color  = "#FFE66D";

// إضافة حدث عند الضغط
btn.addEventListener("click", function() {
  title.textContent = "تم الضغط على الزر!";
  title.style.color = "#FF6B35";
});

// إضافة/حذف class
title.classList.add("active");
title.classList.toggle("hidden");
💡 querySelector يقبل أي CSS selector مثل ".card" أو "#nav"، وهو أقوى من getElementById.
03 تطبيقي

بناء مكوّن تفاعلي كامل

سنبني عداداً تفاعلياً (Counter) من الصفر يزيد وينقص ويُعيد التعيين. هذا المثال يجمع كل المفاهيم: اختيار العناصر، التعامل مع الأحداث، تحديث الـ DOM، والمنطق الشرطي – كل ما تحتاجه لبناء أي مكوّن تفاعلي.

JavaScript – Counter Component 
// HTML المطلوب:
// <div id="counter">0</div>
// <button id="dec">-</button>
// <button id="inc">+</button>
// <button id="rst">إعادة</button>

const display = document.getElementById("counter");
const decBtn  = document.getElementById("dec");
const incBtn  = document.getElementById("inc");
const rstBtn  = document.getElementById("rst");

let count = 0;

function updateDisplay() {
  display.textContent = count;
  display.style.color = count > 0
    ? "#4ECDC4"
    : count < 0
    ? "#FF6B35"
    : "#ffffff";
}

incBtn.addEventListener("click", () => {
  count++;
  updateDisplay();
});

decBtn.addEventListener("click", () => {
  count--;
  updateDisplay();
});

rstBtn.addEventListener("click", () => {
  count = 0;
  updateDisplay();
});
💡 Arrow functions (() =>) هي طريقة كتابة Functions أحدث وأقصر – استخدمها في Callbacks مثل addEventListener.

الأسئلة الشائعة

كل ما يدور في بالك

أجوبة صادقة وواضحة على الأسئلة الأكثر شيوعاً بين المبتدئين.

ابدأ بالترتيب: HTML أولاً لتتعلم هيكل الصفحات، ثم CSS للتصميم، ثم JavaScript للتفاعل. لا تقفز للـ Frameworks مثل React قبل إتقان هذه الثلاثة – الأساس هو كل شيء. منصتنا مصممة تحديداً لهذا الترتيب المنطقي.

بتخصيص ساعة إلى ساعتين يومياً: HTML + CSS في 3-4 أسابيع، JavaScript الأساسيات في 2-3 أشهر، وبناء مشروعك الأول في 4-6 أشهر. الثبات والتطبيق اليومي أهم من عدد الساعات.

لا! تطوير الويب من أسهل المجالات البرمجية للمبتدئين. لا تحتاج رياضيات متقدمة ولا خلفية تقنية. تحتاج فقط جهاز وإنترنت ومتصفح وفضول حقيقي للتعلم – هذه المنصة تتولى الباقي.

تخيّل بناء بيت: HTML هو الهيكل (الجدران والأسقف والأبواب)، CSS هو الديكور والطلاء والأثاث، وJavaScript هو الكهرباء والسباكة – كل ما يجعل البيت "يعمل" ويتفاعل مع ساكنيه.

بالتأكيد! تطوير الويب من أعلى المجالات طلباً في سوق العمل العالمي. بعد إتقان HTML وCSS وJavaScript، يمكنك العمل كـ Front-End Developer، فريلانسر، أو تأسيس مشاريعك الخاصة. الشركات تبحث عن محفظة مشاريع (Portfolio) أكثر من الشهادات.

نعم! المحرر التفاعلي في المنصة يعمل بالكامل في المتصفح بدون خوادم أو إنترنت. بمجرد تحميل الصفحة، يمكنك كتابة الكود وتشغيله لحظياً. مبني بـ Vanilla JavaScript خالص لضمان السرعة والخفة.
